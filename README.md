## Development

#### Clone the repository

    https://github.com/Paul-Cavain/myPortifolio.git

#### Install Dependencies
    npm install
    
    # or
    
    yarn install

#### Start the development server
    npm start

    # or

    yarn start

#### Build for production
    npm run build

    # Or using Yarn

    yarn build
